# Harnais agentique

Ce dossier trace les demandes et les evolutions produit/techniques.

## Regle

Pour chaque demande, creer un fichier dans `agentic/requests/` avec le nom:

`YYYY-MM-DD_slug-court.md`

## Sections obligatoires

- `Besoin`
- `Roadmap`
- `Implementation`
- `Validation`

## Process rapide

1. Creer le fichier de demande.
2. Remplir `Besoin` et `Roadmap` avant de coder.
3. Mettre a jour `Implementation` au fil des changements.
4. Mettre a jour `Validation` apres build/tests manuels.
